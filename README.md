# 🌐 Web Development Portfolio

Welcome to my portfolio website! This project is a showcase of my journey as a web developer, featuring my skills, projects, and contact information.

## 🚀 About the Project
This is a beginner-friendly portfolio website built to demonstrate my frontend development capabilities. I focused on clean code, modern design.

## ✨ Features
- **Project Showcase:** A section dedicated to the websites I've built.
- **Skills Section:** Highlighting my proficiency in various technologies.
- **Contact Form:** A way for viewers to get in touch with me.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling, Flexbox, and Grid for layouts.
